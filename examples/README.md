# Examples

This directory contains example implementations and scenarios for the 6G OPENRAN system.

## Basic Setup

Location: `examples/basic-setup/`

Simple examples to get started:
- Single UE registration
- Basic PDU session establishment
- Simple data transfer

## Advanced Scenarios

Location: `examples/advanced-scenarios/`

Complex use cases demonstrating:
- Multi-slice deployment
- RIC xApp development
- Network optimization
- Edge computing integration

## Running Examples

```bash
# Navigate to example directory
cd examples/basic-setup

# Follow README in each example directory
cat README.md
```

See individual example directories for detailed instructions.
